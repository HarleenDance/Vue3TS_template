# vue3x-vant3x-mobile
* vue3x+vant3x+typescript+axios技术栈; 
* 适配px转vm,设计稿750px; 
* axios请求；
* config/env.testing.js 配置目录vueVantDemo，解决测试环境文件夹非根目录问题；
* 多环境动态API配置

[Vue3x官方文档](https://v3.cn.vuejs.org/)
[Vant3x官方文档](https://vant-contrib.gitee.io/vant/v3/#/zh-CN)
